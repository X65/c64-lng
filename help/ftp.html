usage: ftp host

Ftp implementes the file tranfer protocol used in TCP/IP networks. Using ftp you can download binaries or view text located on a remote host. The host must be identified by its IP-number since DNS lookups are not yet implemented. Ftp supports the following commands:

cd - change directory

pwd - view current directory

type i/a - set ascii (a) or binary (i) tranfer mode

dir - list files in current directory

more - display remote textfile

get - download remote file

quit - leave ftp
