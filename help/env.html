Usage: env

Env print the environment settings as they are inherited from the parent process (the shell you used to invoke env). Currently the only environment variables are:

current working device

Default device for loading/saving files (major: 2=floppy, minor: 8=drive with secundary address 8)

terminal width

Width of the used output terminal. This information is used by applications like "more", that need to know the size of the visible screen.
