usage: date [...]

Options:

-t hh:mm:ss.tt set time

-d ccyy.mm.dd set date

-w ww set weekday

-z shhmm set timezone, s is + or -

-u print tie in unix format

-n print time in ntp format

Set date and time of the real time clock. You have to install a RTC module first - for example ciartc, which emulates a RTC using the TOD-interrupt of CIA1.
