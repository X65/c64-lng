Usage: lsmod

Lsmod lists the installed kernel modules. Each printed line is associated to one module.

Typ

Code consisting of 3 characters that identifies the type of module (for example "SER" for serial interfaces)

SZ

Size of the module - number of allocated pages of internal memory.

Ver

Version number

Num

Number of available virtual devices - a single kernel module might interface more that one device of the same kind.

Addr

Base address of the module's code.
